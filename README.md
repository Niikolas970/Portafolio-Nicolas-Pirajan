# 🚀 Sobre el proyecto

Este repositorio contiene la segunda versión de mi portafolio profesional, desarrollado con **React + Vite** y diseñado para representar mi crecimiento como desarrollador Full Stack.

El proyecto fue construido siguiendo principios de escalabilidad, reutilización de componentes y buenas prácticas de desarrollo Frontend, buscando ofrecer una experiencia moderna, rápida y visualmente atractiva.

## 🎯 Objetivos

- Construir una arquitectura limpia y mantenible.
- Crear componentes reutilizables.
- Aplicar diseño moderno inspirado en **Liquid Glass**.
- Optimizar rendimiento y accesibilidad.
- Mostrar proyectos y habilidades de forma profesional.
